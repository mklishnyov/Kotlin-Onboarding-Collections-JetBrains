На каждом этапе вы можете запускать текущую версию приложения. Однако, если какая-то функциональность ещё не готова, некоторые кнопки могут не работать, а часть информации может не отображаться.

Чтобы запустить приложение, вам нужно запустить функцию `main` внутри файла [TamagotchiApplication.kt](psi_element://org.jetbrains.kotlin.course.tamagotchi.TamagotchiApplicationKt#main):

![Как запустить приложение](../../utils/src/main/resources/images/tamagotchi/ide/run.png)

Пожалуйста, не забудьте _остановить все другие запуски_ нажатием на красную квадратную кнопку:

![Как остановить приложение](../../utils/src/main/resources/images/tamagotchi/ide/stop.png)

Далее нужно открыть любой браузер (мы рекомендуем использовать [Google Chrome](https://www.google.com/chrome/) для отображения элементов, как в примерах) и открыть http://localhost:8080/. Вы увидите главную страницу приложения.

<div class="hint" title="Нажмите, чтобы увидеть, как выглядит главная страница приложения">

![Главная страница приложения](../../utils/src/main/resources/images/tamagotchi/states/main_page.png)

</div>

<div class="hint" title="Нажмите, если отображается приложение с последнего запуска">

Если при запуске игры на экране отображается приложение от предыдущего запуска, вам нужно сбросить кеш. Это обычно можно сделать с помощью сочетания клавиш: `ctrl` + `shift` + `R` (`command` + `shift` + `R` для macOS).
</div>
