В Kotlin вы можете работать с коллекциями по-другому. Вы можете выполнять не только базовые операции, такие как сортировка или поиск некоторых элементов, но и более сложные. Одна из таких операций — группировка элементов по условию.

Самый простой способ сгруппировать элементы — использовать встроенную функцию [`groupBy`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin.collections/group-by.html).

Эта функция позволяет вам сгруппировать элементы исходной коллекции по ключу, возвращаемому заданной функцией `keySelector`, которая применяется к каждому элементу и возвращает карту, где каждый ключ группы связан со списком соответствующих элементов.

Проще говоря, если вы группируете объекты, например, по их длине, то ключами в полученной карте будут возможные значения длины, найденные в исходной коллекции, и для каждого такого ключа будет составлен список объектов из исходной коллекции с данной длиной.

Рассмотрим пример:
```kotlin
fun main() {
    val words = listOf("a", "abc", "ab", "def", "abcd")
    words.groupBy { it.length }.map { (length, words) ->
        val wordsStr = words.joinToString(", ")
        println("Current length is $length. The words: $wordsStr")
    }
}
```

Поскольку мы группируем строки в списке по их длине, этот код выведет в консоль следующий результат:
```text
Current length is 1. The words: a
Current length is 3. The words: abc, def
Current length is 2. The words: ab
Current length is 4. The words: abcd
```
