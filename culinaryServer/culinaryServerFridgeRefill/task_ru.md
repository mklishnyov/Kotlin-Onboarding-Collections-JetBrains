Чтобы начать «готовить», нам нужно пополнить холодильник овощами.  
В этой задаче мы сделаем кнопку `Refill` функциональной. Она должна удалять все старые овощи и добавлять новые.

Объект `FridgeImpl` содержит изменяемый список `vegetables` для хранения овощей.  
В этой задаче вам нужно реализовать два метода внутри `FridgeImpl`:

- `generateRandomVegetables` - метод должен создавать список объектов `Vegetable` по следующим правилам:

  - генерировать `RANDOM_VEGETABLES_NUMBER` случайных _неразрезанных_ овощей с случайным значением свойства `isFresh`
  - генерировать `RANDOM_FRESH_VEGETABLES_NUMBER` случайных _неразрезанных_ несвежих овощей (с установленным свойством `isFresh = True`)

- `refill` - метод должен очищать весь список `vegetables`,
  генерировать новые овощи с использованием функции `generateRandomVegetables` и добавлять их в список `vegetables`
