На каждом этапе вы можете запускать текущую версию приложения.  
Однако, если какая-то функциональность еще не готова,  
то некоторые кнопки могут не работать, а часть информации может не отображаться.

Чтобы запустить приложение, необходимо выполнить функцию `main` внутри файла  
[MasterChefApplication.kt](psi_element://org.jetbrains.kotlin.course.culinary.MasterChefApplicationKt#main):  

![Как запустить приложение](../../utils/src/main/resources/images/master/chef/ide/run.png)

Пожалуйста, не забудьте _остановить все остальные запущенные процессы_, нажав на красную квадратную кнопку:  

![Как остановить приложение](../../utils/src/main/resources/images/master/chef/ide/stop.png)

Далее, необходимо открыть любой браузер (мы рекомендуем использовать [Google Chrome](https://www.google.com/chrome/) для корректного отображения элементов, как в примерах)  
и открыть http://localhost:8080/. Вы увидите главную страницу приложения.  

<div class="hint" title="Нажмите, чтобы посмотреть, как выглядит главная страница приложения">  

![Главная страница приложения](../../utils/src/main/resources/images/master/chef/states/main_window.png)  

</div>

<div class="hint" title="Нажмите, если отображается приложение с последнего запуска">  

Если при запуске игры на экране отображается приложение с предыдущего запуска,  
необходимо очистить кеш. Обычно это можно сделать с помощью комбинации клавиш: `ctrl` + `shift` + `R` (`command` + `shift` + `R` для macOS).  
</div>
