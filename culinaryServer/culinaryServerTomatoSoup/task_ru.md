Давайте практиковаться! Начнем с привычного подхода с коллекциями и приготовим томатный суп из предыдущего урока, используя _список_ томатов.  
В этом задании мы сделаем кнопку `Soup` активной.

Вам нужно реализовать две функции в файле `TomatoSoup.kt`:

- `getTomatoesForSoup` - которая получает `NUMBER_OF_TOMATOES` томатов из _холодильника_ (объекта типа `VegetableType` с типом `VegetableType.Tomato`), посмотрите класс `FridgeImpl`, чтобы найти необходимый API.  
- `prepareTomatoes` - которая принимает _список_ томатов и выполняет следующие действия для каждого элемента из списка:

  - помещает его на кухню (см. `KitchenImpl.put`),
  - режет каждый элемент (см. `KitchenImpl.cut`),
  - берет каждый нарезанный томат (см. `KitchenImpl.take`),
  - кладет каждый томат в кастрюлю (см. `PotImpl.put`).

**Примечание:** Если в холодильнике недостаточно томатов, нажмите кнопку `Refill` в приложении, чтобы пополнить холодильник и иметь возможность приготовить томатный суп.
