В этой задаче вам нужно реализовать функции, которые позволят вам
добавлять уток в магазин уток.

### Задача

Реализуйте три версии функции `addRandomDuck` из класса `GameChangeFunctionsService` в пакете
`org.jetbrains.kotlin.course.duck.shop.functions.change`. Эти функции должны добавлять случайную утку из списка, множества или карты.

Обратите внимание, что вам нужно _действительно_ добавить новую утку, что означает, что в случаях с множеством и картой, вам необходимо сгенерировать утку, которой нет в коллекции, и добавить её.

После выполнения этой задачи кнопка `Add` во всех трех режимах станет активной.

### Подсказки

<div class="hint" title="Нажмите, чтобы узнать, как добавить новый элемент в список или множество">

Чтобы добавить новый элемент, вы можете использовать встроенную функцию `add`:
```kotlin
val listOfNumbers = mutableListOf(1, 2, 3)
listOfNumbers.add(4)
println(listOfNumbers) // 1, 2, 3, 4

val setOfNumbers = mutableSetOf(1, 2, 3)
setOfNumbers.add(4)
println(setOfNumbers) // 1, 2, 3, 4
```

Обратите внимание, что если вы попытаетесь добавить _тот же_ элемент в множество, он не будет добавлен.
</div>

<div class="hint" title="Нажмите, чтобы узнать, как добавить новый элемент в карту">

Вы можете связать новый ключ с новым значением:
```kotlin
val mapOfNumbers = mutableMapOf(1 to "one", 2 to "two")
mapOfNumbers[3] = "three"
println(mapOfNumbers) // 1 to "one", 2 to "two", 3 to "three"
```

В этом случае, если вы попытаетесь использовать существующий ключ, старое значение будет заменено новым.

Если вам нужно добавить что-то только если это отсутствует, вы можете использовать встроенную функцию `putIfAbsent`:
```kotlin
val mapOfNumbers = mutableMapOf(1 to "one", 2 to "two")
mapOfNumbers.putIfAbsent(3, "three")
mapOfNumbers.putIfAbsent(3, "three") // Не будет добавлено
println(mapOfNumbers) // 1 to "one", 2 to "two", 3 to "three"
```
</div>

<div class="hint" title="Нажмите, чтобы узнать, как найти разницу между двумя списками">

Вы можете использовать встроенную функцию `minus` для более эффективного вызова и преобразования одного из аргументов в множество:
```kotlin
val listOfNumbers1 = listOf(1, 2, 3, 4, 5)
val listOfNumbers2 = listOf(1, 2, 3)
println(listOfNumbers1.minus(listOfNumbers2.toSet())) // 4, 5
```

Функция `minus` может работать с коллекциями, это означает, что код `listOfNumbers1.minus(listOfNumbers2)` также корректен, 
но для лучшей производительности вам следует преобразовать список в множество, чтобы работать с меньшим количеством элементов.
</div>
