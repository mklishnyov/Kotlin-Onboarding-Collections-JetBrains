В этом задании вам нужно реализовать функцию, которая способна разделить уток в Магазине Уток на две категории — те, у которых есть что-то, связанное с Kotlin, и те, у которых этого нет — и упорядочить их согласно этому условию.

### Задание

Реализуйте функцию `divideDucksIntoKotlinAndNonKotlin` из класса `GameActionFunctionsService` в пакете `org.jetbrains.kotlin.course.duck.shop.functions.action`. Эта функция должна разделить уток на две группы — те, у которых есть атрибут Kotlin (поле `hasKotlinAttribute` равно `true`), и тех, у кого его нет — и вернуть эти две коллекции.

У нас есть только одна функция, поскольку функция `fun Collection<Duck>.divideDucksIntoKotlinAndNonKotlin()` работает для обеих коллекций — для `List` и для `Set`. Для коллекции типа map эта функция не существует.

После выполнения этого задания кнопка `Partition` в режимах `list` и `set` станет активной.

### Подсказки

<div class="hint" title="Нажмите, чтобы узнать, как разделить коллекцию на основе условия">

Вы можете сделать это несколькими способами, но самый простой — использовать встроенную функцию [`partition`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin.collections/partition.html). Она принимает предикат (условие) для разделения коллекции на две части согласно этому условию:

```kotlin
val listOfNumbers = listOf(1, 2, 3, 4, 5, 6)
println(listOfNumbers.partition { it % 2 == 0 }) // Возвращает пару из двух списков: [2, 4, 6], [1, 3, 5]
```
</div>
